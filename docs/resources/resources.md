## :brain: Programming Resources

- [Odin Project](https://www.theodinproject.com/) - A free, full-stack web development curriculum that emphasizes hands-on projects and real-world portfolio building.

- [Rosalind](https://rosalind.info/problems/locations/) - An interactive platform for learning bioinformatics and computational biology through problem-solving challenges.

- [freeCodeCamp](https://www.freecodecamp.org/learn) - A nonprofit platform offering free coding lessons, certifications, and practical projects across web development and computer science topics.

- [ossu (Open Source Societ University)](https://github.com/ossu) - A community-driven, self-paced computer science curriculum built entirely from free online resources.

- [PortSwigger Web Security Academ](https://portswigger.net/web-security) - A hands-on web security training platform that teaches vulnerability discovery and exploitation techniques through interactive labs.
