# Purpose :goat:
The purpose of this section is to document learning new programming languages

- [C#](c_sharp.md) :knife:
    - [C# language reference](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/?utm_source=chatgpt.com)
    - Object-oriented language built on the .NET ecosystem for web, desktop, cloud, and enterprise applications.
    - Strong support for asynchronous programming (`async/await`), LINQ, and memory management through the .NET runtime.
    - Commonly used with ASP.NET Core, Entity Framework Core, REST APIs, and microservices.

- [Java](java_file.md) :coffee: 
    - [Java Programming Language](https://docs.oracle.com/javase/8/docs/technotes/guides/language/)
    - Object-oriented, platform-independent language running on the Java Virtual Machine (JVM).
    - Common choice for enterprise applications, backend services, Android development, and distributed systems.
    - Strong ecosystem with Spring Boot, Maven/Gradle, multithreading, and robust garbage collection.

- [Python](python_file.md) :snake:
    - [Python Tutorial](https://docs.python.org/3/tutorial/?utm_source=chatgpt.com)
    - High-level, dynamically typed language focused on readability and rapid development.
    - Widely used for automation, scripting, data science, machine learning, AI, and backend development.
    - Extensive ecosystem with libraries such as NumPy, Pandas, TensorFlow, PyTorch, and FastAPI.

- [Rust](rust.md) :wrench:
    - [Rust Standard Library](https://doc.rust-lang.org/std/)
    - Systems programming language emphasizing memory safety without a garbage collector through its ownership model.
    - Delivers high performance suitable for operating systems, embedded systems, networking, and performance-critical software.
    - Provides fearless concurrency, strong compile-time guarantees, and excellent tooling with Cargo.
